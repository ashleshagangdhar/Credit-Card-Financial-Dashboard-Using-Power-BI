# Credit-Card-Financial-Dashboard-Using-Power-BI
Developed an interactive Power BI dashboard to analyze credit card transactions and customer behavior. Created KPI-driven visualizations for revenue, spending patterns, customer segments, and transaction trends. Used Power Query, DAX, and data modeling to generate actionable business insights.
## Tools Used
- Power BI
- Power Query
- DAX
- SQL
- Excel

## Key Features
- Revenue Analysis
- Customer Segmentation
- Transaction Trend Analysis
- Spending Pattern Analysis
- Interactive Filters and Slicers
- KPI Tracking Dashboard
